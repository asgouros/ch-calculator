Calculator

Description

The goal of this challenge is to built a Javascript class in the form of an object that will serve as a calculator. The calculator should contain simple functions to perform the following actions:
  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Power
You should display the result in the console. For each function your code should check the following: All arguments are required. If an argument is missing, throw an Error. Check if the action makes sense (i.e. division with 0 is wrong)

Implementation notes

The class functionality is presented by its tests contained in Calculator.spec.js
To run them please use the Tests tab and hit Run all tests.
